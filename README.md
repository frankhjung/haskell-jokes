# Jokes

Read a Joke from <https://official-joke-api.appspot.com/>.

This is an example program showing how to all an API and parse the JSON response.

```json
{
  "type": "general",
  "setup": "What do you call a pig with three eyes?",
  "punchline": "Piiig",
  "id": 216
}
```
